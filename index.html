Acerca de la aplicacion


Esta aplicacion web le permitira buscar libros usando la API de Google Books.

Para bucar libros debe ir a la seccion "Libros" de la aplicacion usando el menu.

Puede realizar busquedas usando solo el titulo o tambien puede filtrar aun mas la busqueda ingresando un autor y/o editorial.

Tambien puede ver el historial de sus busquedas presionando el boton "Historial".

Para ver mas informacion acerca de la aplicacion puede hacerlo desde la seccion "Contacto".
